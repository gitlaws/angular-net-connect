Credit Card Input Component

This Angular component simplifies credit card data entry by:

  • Formatting: Automatically formats card numbers as they are typed.
  • Identifying: Recognizes the card network (Visa, Mastercard, etc.) based on the initial digits.
  • Validating: Performs basic card number checks and communicates with an authentication service for advanced validation.
  • User Interaction: Simply type the credit card number into the input field. The component handles the rest, providing visual feedback and error messages as needed.
