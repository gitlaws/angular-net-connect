This component helps users create strong and secure passwords by providing real-time feedback. As the user types their password, a progress bar dynamically updates to reflect how well the password meets the following security requirements:

  1. Minimum Length: 8 characters
  2. Contains at Least One Uppercase Letter
  3. Includes at Least One Number
  4. Contains at Least One Special Character

The progress bar visually indicates which criteria have been met, guiding users to create a compliant and strong password.

  • Min Length of 8 Required.
  • One or More Capital Letter Required.
  • One or More Number Required.
  • One or More Special Characters Required.
