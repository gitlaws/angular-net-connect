Credit Card Input Validator

This component provides a user interface for entering and validating a credit card number. It includes an input field for the credit card number, dynamically displays the card type icon (Visa, MasterCard, American Express, Discover, or a generic credit card icon), and shows validation messages indicating whether the card number is valid and its formatted version. The component also supports read-only mode and ensures the input adheres to a maximum length of 16 characters.

Note: While these are common starting digits, there are many variations within each network. Credit card validation and authorization is a multi-layered process involving several checks and interactions with different security systems.
